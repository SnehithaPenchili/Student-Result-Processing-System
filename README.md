# Student-Result-Processing-System
This project implements a Student Result Processing System in C language.
The program reads student details from a file, validates the data, computes results, assigns grades and CGPA, and finally displays a detailed tabular report along with class statistics.

The system is designed using modular programming, structures, and file handling, as part of  academic coursework.
The commands to run the program is :
gcc main.c -o output

./output

The final ouput is displayed on the terminal
